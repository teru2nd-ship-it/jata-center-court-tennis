# Tetromino Drop Sample

テトリス風パズルのプレイ可能サンプル。

## 遊び方

- 左右: `ArrowLeft` / `ArrowRight`
- 下移動: `ArrowDown`
- 回転: `Z` / `X` / `ArrowUp`
- 一気に落とす: `Space`
- 一時停止: `P` または `Pause` ボタン
- スマホ: 画面下のボタン

## 実装済み

- 10 x 20 の落ち物パズル盤面
- 7種類のテトロミノ風ブロック
- 次ブロック表示
- ゴースト位置表示
- ライン消去、スコア、レベル
- ハイスコア保存
- リスタート、一時停止
- モバイル用タッチ操作

## 確認方法

`index.html` をブラウザで開く。

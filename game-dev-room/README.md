# Game Dev Room

TERUさんのゲーム開発まわりの共通作業室です。

## 目的

- 既存ゲームの改善メモを集約する
- 新規ゲーム案を育てる
- 共通UI、音、シェア導線などを横断的に整える
- 公開前チェックと外部変更ルールを明確にする

## 最重要ルール

外部変更が出る作業は、実行前にTERUさんへ確認します。

対象例:

- GitHub push
- GitHub Issue作成
- Vercel Preview確認後の本番反映
- 公開URLや本番設定の変更
- 公開済みゲームの差し替え

## 進め方

1. 作業前にこのフォルダと `handover/2026-05-08_project_handover.md` を確認する。
2. 既存ゲーム改善は `projects/` の該当ゲームに追記する。
3. ゲームをまたぐ改善は `shared-improvements.md` に追記する。
4. 新規ゲーム案は `new-game-ideas.md` に追記する。
5. push、Issue作成、本番反映などはTERUさん確認後に実行する。

# 新規ゲーム案

## 候補

### クロスワードパズル

言葉遊び系。問題作成の仕組みが必要。

### なぞなぞ

短時間で遊べる。問題データを増やしやすい。

### ロードランナー的な単純アクションゲーム

レトロ寄りの操作感。ステージ設計が楽しさの中心になる。

### 車好きな息子向けの車運転系ゲーム

子ども向け。シンプル操作、車種、音、コースの楽しさが大事。

### マイクラの建築のみに特化したシンプルゲーム

作る楽しさに寄せる。戦闘や複雑なサバイバル要素は入れすぎない。

### 豆知識クイズ

ブログや日々の学びとも相性が良い。問題追加しやすい形にする。

### テトリス風パズル

落ち物パズル。まずはプレイ可能サンプルとして [Tetromino Drop Sample](prototypes/tetromino-drop/) を作成済み。
デザイン、音、シェア、スマホ操作感は後で詰める。

## 優先順位を決める観点

- すぐ試作できるか
- TERUさんや家族が遊びたくなるか
- 問題やステージを増やしやすいか
- 既存ゲームの共通UIを流用できるか
- スマホで遊びやすいか
